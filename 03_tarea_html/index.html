introducción al lenguaje

Html es el lenguaje basico para crear paginas web, ya que organiza el contenido mediante la etiqueta o imagenes y enlaces.

Etiquetas de formato

Las etiuetas "hr" y "br" en HTML se utilizan para organizar el contenido de una pagina web. la etica "hr" sirve para insertar una linea horizontal que separa secciones o temas, mientras que "br" se usa para hacer un salto de linea de un texto sin iniciar un nuevo parrafo
